# Ciphera

## Premium Archive Extraction Utility

A powerful archive extraction utility for macOS with cloud sync capabilities.

### Features
- Multi-format support: Zip, Rar, 7z, tar, gz, bz2, xz, iso, dmg
- Password-protected archive extraction
- Cloud sync with Supabase
- Organized Watch/Trash/Failed output
- Real-time progress display
- Command-line and interactive mode

### Installation
1. Download and run Ciphera.app
2. Or use: bash Ciphera.sh -s -p -t

### Supabase Setup
Run supabase/schema.sql in your Supabase SQL editor.
